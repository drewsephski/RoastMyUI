Privacy Policy

Effective date: November 26 2025

This privacy policy explains what data the Roast My UI Chrome extension ("the Extension") collects, how it is used, and how it is protected.

Data‑Usage Disclosure (as submitted to the Chrome Web Store)

Data Category Collected Now? Collected In‑Future? (with explicit opt‑in) Explanation
Personally identifiable information No No The extension never asks for or stores any PII.
Health information No No No health‑related data is relevant to the extension’s purpose.
Financial and payment information No No The extension is free and does not handle payments.
Authentication information No No No login or authentication is performed inside the extension.
Personal communications No No The extension never reads or transmits personal communications.
Location No Possible – only if a future “regional‑settings” feature is added and the user explicitly enables it. Currently we do not collect any location data.
Web history No No We only access the active tab when the user clicks the extension; we do not record a browsing history.
User activity Limited – the click that launches the extension is required. Possible – optional analytics events may be sent if the user opts‑in via a settings toggle. No continuous monitoring; only a single activation event is captured.
Website content Yes Possible – future “save‑roast‑report” feature could store a copy of the report on the server, but only with explicit user consent. Required to feed the AI “roast” engine with the page’s content (URL, HTML/Screenshot). Data is held in memory only for the duration of the request and discarded immediately after the response is returned.

How We Protect Your Data

  • All communication between the extension and our backend AI service is encrypted with HTTPS.
  • Data is processed in a stateless manner – we do not keep persistent logs of page content or URLs.
  • Only the minimal information required to generate a roast is transmitted.

Future Data Collection

If we later add features such as usage analytics, error reporting, or personalized settings sync, we will:

  1. Ask for explicit permission via a new optional setting in the extension UI.
  2. Update this privacy policy on the Chrome Web Store and on our website.
  3. Provide a clear opt‑out mechanism for users.

Your Rights

You may request deletion of any data you have voluntarily submitted (e.g., feedback) by contacting us at privacy@roastmyui.com. We will respond within 30 days.

Contact Us

If you have any questions about this privacy policy, please email us at privacy@roast-my-ui.com.
