# Chrome Web Store Assets

This folder contains all the promotional images required for publishing to the Chrome Web Store.

## Files Included

### Screenshots (1280x800)

- `screenshot-1.png` - Main extension popup interface
- `screenshot-2.png` - Extension analyzing a website (low score example)
- `screenshot-3.png` - Extension analyzing a website (high score example)

### Promotional Tiles

- `small-promo-tile.png` - 440x280 small promotional tile
- `marquee-promo-tile.png` - 1400x560 marquee promotional banner

## Usage

When submitting to the Chrome Web Store:

1. **Screenshots**: Upload at least one (up to 5) in the "Store listing" section
2. **Small promo tile**: Upload in the "Promotional images" section
3. **Marquee promo tile**: Upload in the "Promotional images" section

All images are 24-bit PNG format without alpha channel, as required by Chrome Web Store.
